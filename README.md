Создана демо-страница для создания и редактирования списка книг.

Редактирование списка реализовано внутри каждой книги. Поля атвор и книга редактируются через ввод текста, обложка книги через кнопку добавления новой обложки.

Все книги сохраняются в локальное хранилище и доступны после перезагрузки страницы.

Удаление проиходит по нажатию на сответствующую кнопку.

Для сохранения измененной книги в локальное хранилище нажать соответствующую кнопку.
